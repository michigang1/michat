## General Information
- The application is a chat messenger. 
- The app uses registeration by password/emai.
- My app allows people to communicate with each other without any restrictions (technical limitations do not count :) )



## Technologies Used
- IDE: Android Studio
- Language: Kotlin
- Other instruments:
   - Google Firebase
   - XML
   - jUnit
